# 親師聯絡簿 V7 海洋晨曦正式修正版

## 本版更新
- 採用使用者選定的海洋晨曦、飛魚與音符風格
- 登入身份改為船錨（教師）、木質船舵（家長）、燈塔意象（訪客）
- 合併重複訪客入口，只保留「訪客體驗」
- 座號必填；學生姓名可空白並後續補填
- 支援一次新增多人，格式可只輸入座號或「座號,姓名」
- 教師可後續補填家長姓名與聯絡電話
- 保留家長問答、教師回覆、簽閱與完整資料匯出
- 匯出帳密時包含家長姓名與聯絡電話
- 不含校名、地名或特定學校文字，可分享給其他學校使用
- 右下角署名為 Meng

## 部署
解壓後將所有檔案覆蓋 GitHub 專案，再執行：

```bash
cd ~/parent-contact-book
git pull
firebase deploy --only hosting,firestore:rules
```
